# Brief originale (ricevuto dall'utente)

## 1. Company name and blurb
**Nome:** Bibliò

**Blurb/Descrizione:**
Bibliò è una piattaforma e-commerce culturale italiana che rivoluziona il mercato editoriale attraverso un modello ibrido: acquisto di libri fisici e digitali, noleggio flessibile per 30 giorni, e un consulente AI integrato (MyBibliò) basato su Claude che guida ogni lettore verso la scelta perfetta eliminando la "paralisi della scelta".

## 2. Design System Overview

### Colori Principali
- **Primary Red/Coral:** #C75550 (CTA, accenti, heading)
- **Dark Brown/Charcoal:** #2A2A2A (Testo body, sfondo scuro)
- **Cream/Off-white:** #F5F1E8 (Sfondo principale, card)
- **Blue Navy:** #3A4A5C (Categorie, icone)
- **Light Gray:** #E8E3DB (Bordi, separatori)

### Tipografia
- Headings: Serif (Lora, Playfair Display)
- Body: Sans-serif (Inter, Outfit)
- Tono: Eleganza editoriale, minimalista, accessibile

### Abbonamento
- Bibliò Plus: 9,99€/mese. 2 noleggi mensili, spedizione gratuita, rimozione pubblicità.

### Stack
- Frontend: WordPress + WP-Claude-Interface (plugin)
- AI: Claude 4.5 Haiku via Anthropic API
- Data: Open Library API + Google Books API
